# 0POO 🚀

Repositorio dedicado al estudio y práctica de la **Programación Orientada a Objetos (POO)**.

## 🛠️ Conceptos Clave
* **Abstracción:** Modelado de entidades del mundo real.
* **Encapsulamiento:** Protección y jerarquía de datos.
* **Herencia:** Reutilización de código mediante clases.
* **Polimorfismo:** Flexibilidad en la respuesta a métodos.

## 💻 Instalación y Uso

Para clonar el repositorio en tu máquina local (Windows):

1.  Abre la terminal (PowerShell o CMD).
2.  Ejecuta el siguiente comando:
    ```bash
    git clone [https://github.com/Seggov/0POO.git](https://github.com/Seggov/0POO.git)
    ```
3.  Entra a la carpeta del proyecto:
    ```bash
    cd 0POO
    ```
